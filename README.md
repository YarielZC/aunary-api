# Aunary - API Backend

Este repositorio contiene la API REST y la lógica de negocio de **Aunary**, una plataforma diseñada para conectar creadores de ideas con colaboradores técnicos. El backend está optimizado para funcionar en entornos Serverless.

## 🛠 Stack Tecnológico
- **Lenguaje:** Python
- **Framework:** FastAPI
- **Base de Datos:** PostgreSQL
- **Caché:** Redis
- **Almacenamiento:** Cloudinary
